# Yappinee

A modern chatting and video call streaming application built with JavaScript.

## Overview

Yappinee is a real-time communication platform that enables users to connect seamlessly through instant messaging and high-quality video calling. Whether you're staying in touch with friends, collaborating with teammates, or conducting remote meetings, Yappinee provides a smooth and intuitive experience.

## Features

- **Real-time Messaging** - Send and receive instant messages with lightning-fast delivery
- **Video Calling** - Crystal-clear video streaming for face-to-face conversations
- **User-Friendly Interface** - Clean and intuitive design for effortless communication
- **Cross-Platform Support** - Access your chats and calls from any device

## Tech Stack

- **Language**: JavaScript (98.7%)
- **Frontend**: Modern web technologies for responsive UI
- **Backend**: Real-time communication infrastructure
- **Video Streaming**: High-performance video codec and streaming

## Getting Started

### Prerequisites

- Node.js (version 14 or higher)
- npm or yarn package manager

### Installation

1. Clone the repository:
```bash
git clone https://github.com/Aenansh/yappinee.git
cd yappinee
```

2. Install dependencies:
```bash
npm install
```

3. Configure environment variables:
```bash
cp .env.example .env
# Edit .env with your configuration
```

4. Start the application:
```bash
npm start
```
